This is the source code for [hisbaan.com](https://hisbaan.com)

# Development

Clone the repository:

```bash
git clone git@github.com:hisbaan/hisbaan.com
cd hisbaan.com
```

Install dependencies:

```bash
pnpm install
```

Run dev environment:

```bash
pnpm run dev
```

Build the static site:

```bash
pnpm build
```

Serve the static site:

```bash
pnpm next
```
